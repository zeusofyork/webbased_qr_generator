# QR Code Generator

A simple Python web application to generate QR codes from text or URLs. Built with Flask and the `qrcode` library.

## Features
- Generate QR codes for any input text or URL.
- Download the QR code as a PNG file.

## Requirements
- Python 3.8+
- Flask
- qrcode
- Pillow
